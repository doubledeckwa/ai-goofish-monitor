Xianyu login status collection expanded privacy policy

Version：2026-01-13 · This document describes browser extensions“Xianyu Login State Extractor”How data is collected, used and stored。

single purpose

Extension's only purpose: when the user clicks“When extracting，Read the login status of the current page Cookie、Common request headers and browser environment information are used to generate local JSON For users to copy to Xianyu monitoring crawler for use。

Types of data collected (only when a user clicks“Triggered when "Extract"）

  • Authentication information: Login for the current page Cookie/token。
  • Request header：User-Agent、Accept、Accept-Language、Accept-Encoding、Referer、Sec-CH UA Series field、Origin、Cache-Control/Pragma、Upgrade-Insecure-Requests、Content-Type（If any）。
  • Browser environment information: language、time zone, platform、hardwareConcurrency、deviceMemory、maxTouchPoints、webdriver status, screen size/resolution/color depth、devicePixelRatio、User-Agent。
  • current page URL，used to generate JSON Describe the current site。
  • Store data：localStorage/sessionStorage，Read only current page, length exceeds 4096 Byte keys are discarded and marked。

How data is used

  • Data is assembled in local memory only as JSON，For manual copying by users; will not be uploaded to any server。
  • Not used for advertising or images、Remarketing or any purpose unrelated to the single purpose stated above。

Storage and transmission

  • No persistent storage; no writing to remote services。
  • Clipboard copying is triggered by user active click。

Permission description

  • tabs / activeTab：Only when the user clicks "Extract”When accessing the current tab URL and inject the collection script。
  • scripting：Inject script reading into the current page navigator/screen information、localStorage/sessionStorage。
  • webRequest：Launch once in the current domain name fetch Read the request header and generate JSON，Does not intercept or modify traffic。
  • storage：Save user consent status and function switches, and do not store sensitive data。
  • cookies（If used): read the current page Cookie to generate login status JSON。

User Consent and Control

  • Only check the box in the pop-up window to agree and click "Extract"”The above data will be collected only when it is used; it will not be collected if it is not used.。
  • You can close or uninstall the extension at any time, which will stop all collection activities。

Data sharing

Do not share or sell data with any third parties。
